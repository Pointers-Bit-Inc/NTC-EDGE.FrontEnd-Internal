# Requirements

expo: https://docs.expo.dev/get-started/installation/

yarn: https://classic.yarnpkg.com/en/docs/install#mac-stable

For node, you can use nvm https://github.com/nvm-sh/nvm

# Installation

$ npm install -g expo-cli@4.13.0

$ nvm install 16

$ nvm use 16

 $ yarn install

 $ expo start
